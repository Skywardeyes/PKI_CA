PKI Web 控制台

现代化双链路控制台：Intl 国际算法链路与 GM 国密链路并行编排，支持证书生命周期、mTLS 验证、TLS 观测与审计留痕。

访问控制（可选）

若启动服务时设置了环境变量 PKI_WEB_TOKEN，请在下方填写相同令牌，请求将携带 X-Admin-Token 头。

建议现场演示时将 Intl 与 GM 分链路操作，避免复用同名客户端导致状态相互影响。
